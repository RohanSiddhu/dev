## Build Instructions
### Tools
1. GNU Make
### Dependencies
1. Kernel Development Package
### Build Command
In project's root directory, run: ```make```
### Manage Module
List modules: ```lsmod```

Load module: ```sudo insmod birthday.ko```

Remove module: ```sudo rmmod birthday```

Print kernel ring buffer: ```dmesg```
